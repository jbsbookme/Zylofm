# ZyloFM — Admin Dashboard

Admin web para gestionar biblioteca, subir música, aprobar DJs y probar el assistant.

## URLs

- Producción: `https://admin.jblatinmusic.net`
- API (target): `https://api.zylofm.com`

## Desarrollo local

```bash
npm install
npm run dev
```

## Variables de entorno

- `NEXT_PUBLIC_API_URL` (ejemplo: `https://api.zylofm.com`)
